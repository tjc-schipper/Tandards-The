Getting started

General Information

Thank you for purchasing our theme. We are happy that you are one of our customers and we assure you won't be disappointed. We do our best to produce top notch themes with great functionality, premium designs and human readable code. Before you get started we highly encourage you to get familiar with this documentation file. Spending half an hour reading the manual may save a lot of your time and avoid questions with obvious answers.

This theme is based on Bootstrap v3.3.5 and LESS framework.

If you have any questions that are beyond the scope of this help file, feel free to post them on our support forum at https://ancorathemes.ticksy.com/.

PLEASE NOTE! Our support covers getting setup, trouble using any features, and any bug fixes that may arise. Unfortunately, we cannot provide support for customizations or 3rd party plugins. If you need help with customizations of your theme, then you should ask for help from a developer.

Typography

Image alignment can be changed by adding one of the values into the <figure class=> tag:

<figure class="sc_image sc_image_shape_round pull-left">
<figure class="sc_image sc_image_shape_round pull-right">

Highlight



To highlight an element add one of the style values below into the <span class=> tag:

<span class="sc_highlight sc_highlight_style_1">ipsum dolor</span>

Skills

You can customize the skills bar by changing width, data-stop and data-ed values as well as their styles. Check below screenshot.

<div class="sc_skills_item sc_skills_style_1 odd first inited"> <div class="sc_skills_count" style="border-color: rgb(52, 184, 192); width: 80%; background-color: rgb(52, 184, 192);"></div> <div class="sc_skills_total" data-start="0" data-stop="80" data-step="1" data-max="100" data-speed="22" data-duration="2156" data-ed="">80</div> </div>

Contact form



There are several files responsible for contact form functionality:

root directory.../js/main.js
root directory.../contact.html
root directory.../contact-form.php

Your email address should be specified in the contact-form.php file. See the code below.

$contact_email = 'your@email.com';

Google map



File responsible for Google map functionality:

root directory.../js/map.js

root directory.../js/custom/googlemap_init.js

Address/coordinates should be specified in map.js file.



Flex Slider



Files responsible for the Swiper Slider functionality:

root directory.../js/vendor/jquery.flexslider-min.js
flexslider.css root directory.../css/flexslider.css



Less Framework

As we have mentioned earlier this theme was built with Less Framework. More about this framework can be found here. Corresponding files can be found in your root.../src/less/... directory.

File responsible for variables can be found in this directory: root directory.../src/less/variables.less.
As for the rules, they can be found here: root directory.../src/less/styles.less.

PSD Files

All the related .PSD files can be found in the theme's archive. Otherwise you can always request them by contacting our support team.
